# Detection Pipeline

## Purpose
A modular Detection-as-Code repository designed to version, test, and deploy detection logic across HELK environment.

## Key Features
- Sigma-based rule authoring and conversion
- CI/CD automation for rule validation
- ATT&CK coverage tracking
- Version-controlled detection lifecycle

## Tech Stack
- Sigma CLI
- GitHub Actions
- HELK (Elastic)
- Atomic Red Team (for validation)
